Realiza un seguimiento de los tiempos de luz solar y de luna. Cuenta con una aplicación, reloj despertador y colección de widgets.

Suntimes muestra el amanecer y el atardecer, el crepúsculo (civil / náutico / astronómico), la hora azul y dorada, los solsticios y equinoccios, la salida y puesta de la luna, las fases de la luna y la iluminación. El reloj de alarma programa alarmas (o notificaciones) con o sin repetición diaria para el amanecer, el atardecer y otros eventos. La aplicación incluye muchas funciones comúnmente asociadas con un reloj mundial.

Los tiempos informados por esta aplicación son aproximados. La aplicación no requiere conectividad de red, el GPS es opcional.

La aplicación se beneficia de los siguientes permisos:
* <small><u>ACCESS_FINE_LOCATION</u> </small> se utiliza para obtener la posición actual (latitud, longitud, altitud).
* <small><u>BOOT_COMPLETED</u> </small> se usa para restaurar las alarmas activas después de un reinicio.
* <small><u>SET_ALARM</u> </small> se usa para programar alarmas usando la aplicación AlarmClock.
* <small><u>VIBRATE</u> </small> es usado por notificaciones de alarma.
* <small><u>READ_EXTERNAL_STORAGE</u> </small> se utiliza para reproducir sonidos de alarma ubicados en la tarjeta SD.
* <small><u>WRITE_EXTERNAL_STORAGE</u> </small> se usa para respaldar datos en archivos (lugares, temas, etc.) <small> (api & le; = 18 solamente) </small>.

Nota: La "Integración de calendario" y los permisos agregados en v0.9.0 se eliminaron en v0.10.0. Esta función ahora está disponible como una aplicación complementaria separada; <a href="https://f-droid.org/en/packages/com.forrestguice.suntimescalendars"> Calendarios de Suntimes </a>.
