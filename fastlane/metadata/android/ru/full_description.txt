Отслеживает время солнечного и лунного света. Содержит информационное приложение, будильники и коллекцию виджетов.

Suntimes показывает восход и закат солнца, длину тени, сумерки (гражданские, навигационные, астрономические), режимное время и "золотой час", солнцестояния и равноденствия, восход и заход луны, фазы луны и ее освещенность. Будильники можно настраивать на ежедневные повторяющиеся сигналы (или просто уведомления) для восхода, заката и других событий. Приложение содержит множество функций, связанных с мировым временем.

Время, рассчитываемое приложением, является приблизительным. Приложение не требует подключения к интернету, использование GPS — по усмотрению пользователя.

Приложению может запрашивать разрешения:
&bull; <small><u>ACCESS_FINE_LOCATION</u></small> для получения текущего местоположения (широта, долгота, высота);
&bull; <small><u>BOOT_COMPLETED</u></small> для восстановления активности будильников после перезагрузки устройства;
&bull; <small><u>SET_ALARM</u></small> для настройки сигналов будильника с помощью приложения AlarmClock;
&bull; <small><u>VIBRATE</u></small> для использования вибросигнала в будильниках;
&bull; <small><u>READ_EXTERNAL_STORAGE</u></small> для использования SD-карты как хранилища пользовательских сигналов будильника <small>(только api&le;=18)</small>.
&bull; <small><u>WRITE_EXTERNAL_STORAGE</u></small> для резервного копирования данных (места, темы интерфейса и т. д.) <small>(только api&le;=18)</small>.

Примечание: интеграция с календарями и разрешения, добавленные в приложении версии 0.9.0 были удалены в версии 0.10.0. Эта функция теперь доступна в виде отдельного приложения <a href="https://f-droid.org/en/packages/com.forrestguice.suntimescalendars">Suntimes Calendars</a>.
